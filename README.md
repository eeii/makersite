# iMaker Website

The website for iEE courses. You can see the generated files at [su-da/website](https://github.com/su-da/website/tree/gh-pages) repository.

## Getting started

Install dependencies:

``` bash
$ git clone https://github.com/su-da/website.git
$ cd website
$ npm install
```

Generate:

``` bash
$ hexo generate
```

Run server:

``` bash
$ hexo server
```
